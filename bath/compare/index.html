How to use this page

Start with two or three likely options, compare the practical basics, then move into school profiles and open-day planning.

  1. Select schools above.
  2. Use the toggle to switch between overview and A-level metrics.
  3. Open the profile pages for details and open days.

What this page is best for

This compare tool is designed for first-pass narrowing, not final judgement. It is strongest on factual differences such as age range, boarding format, fees, bursaries and — for senior schools — A-level performance and subject mix.

For fuller context, use the school profiles and open-days guide alongside this page.
